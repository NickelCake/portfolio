# Nicole Lee Portfolio

## Prerequisites
- Node.js + npm (you already have npm installed)

## Install
```bash
npm install
```

## Run locally
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- Legacy case-study pages are served from `public/legacy/` and linked from the “Selected Work” section.
- To enable the resume download button, add your resume at `public/resume.pdf`.
